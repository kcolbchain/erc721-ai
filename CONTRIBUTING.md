# Contributing to erc721-ai

See the main [CONTRIBUTING guide](https://github.com/kcolbchain/meridian/blob/main/CONTRIBUTING.md) — the same process applies to all kcolbchain repos.

1. Browse [open issues](https://github.com/kcolbchain/erc721-ai/issues)
2. Claim by commenting
3. Fork, branch, PR
4. Get reviewed within 48 hours

Issues tagged `bounty` have paid rewards. Ship PRs to unlock paid work at funded blockchain companies.

- Docs: https://docs.kcolbchain.com/erc721-ai/
- Email: research@kcolbchain.com
